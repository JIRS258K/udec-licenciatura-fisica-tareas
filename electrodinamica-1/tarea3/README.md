## Problemas
- **Problema 1:** Esfera conductora de radio $R$ semisumergida en la interfaz entre un líquido de constante dieléctrica $\kappa_1$ y un gas de constante $\kappa_2$, con carga total $Q$. Se resuelve el potencial con la ecuación de Laplace, se obtiene el campo eléctrico y se calculan las densidades de carga de polarización sobre la superficie.
- **Problema 2:** Esfera de radio $R$ con densidad de carga superficial constante $\sigma$ que rota con velocidad angular $\vec\omega$. A partir del potencial vectorial se halla el campo magnético exterior y se representan sus líneas de campo con el script de Python.
