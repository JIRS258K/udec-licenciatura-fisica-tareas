## Problemas
- **Problema 1:** Campo eléctrico de una placa rectangular finita de lados $a$ y $b$ con densidad superficial uniforme. Se obtiene $\vec E(x,y,z)$, se analizan cortes en el plano $xz$, un mapa 3D del campo y el límite de placa infinita.
- **Problema 2:** Carga puntual en el interior de una cavidad esférica tallada en un bloque conductor conectado a tierra. Usando el método de las imágenes se calcula el potencial en todo el espacio, el campo eléctrico y las cargas inducidas sobre la superficie de la cavidad, además de un gráfico del campo y las equipotenciales.
